# GDUFS-Tools

一些自己写的小工具

## 说明

众所周知泥外的教务实在是老，老得只能用IE浏览器才能正常弹出成绩细则窗口。

作为chrome忠实用户就只能写个小程序来查成绩了。

## 注意

目前成绩查询必须连接广外校园网才可用

config.json明文存储账号密码，分享软件时记得清除



可以用PyInstaller自行构建，gradeNG.spec仅供参考，需自行添加ddddocr所需的onnx文件

## 计划
